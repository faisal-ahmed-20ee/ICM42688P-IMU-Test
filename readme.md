# ICM42688P-IMU-Test
Driver and example project for the ICM-42688P 6-axis IMU (accelerometer + gyroscope), targeted at the Silicon Labs xG24-DK2601B development kit.

## Overview
This repository contains the ICM-42688P sensor driver, IMU abstraction layers, and a small example application demonstrating sensor initialization and data acquisition on the Silicon Labs xG24-DK2601B (Simplicity Studio) platform.

## Supported Hardware
- ICM-42688P (InvenSense/TDK)
- Silicon Labs xG24-DK2601B development kit (project configured for this board)
